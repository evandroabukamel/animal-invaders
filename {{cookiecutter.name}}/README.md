This is a started client Unity project with a minimal subset of modules already setup.

- Authentication
- Simple Configuration
- SQL Persistence
- Metagame Base
- Pitaya v2

## Getting Started

### Requirements

* Unity 2019.4.28f1 or later
* Make

### Running the client

In order to run it in your machine, you'll need to first start up the metagame server and its dependencies. 
After cloning the respective metagame server project, make sure you execute the steps for starting the server so the client can connect to it.

Notice there is a minimal server template for generating a minimal metagame server project. For further details, please check this template [here](https://git.topfreegames.com/ring/template/minimal-metagame-template).
